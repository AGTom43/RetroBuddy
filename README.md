# RetroBuddy
A Jamboard Clone

## Installation Instructions

### Frontend

1. Install bun (https://bun.sh/)
2. Run `cd frontend` 
3. Run `bun install`
4. Run `bun run dev` or `vite --host` for multiuser
5. Go to localhost

### Backend

1. Use JDK 23
2. Run services in `local-dev-compose.yml`
3. Run `RetrobuddyApplication` run configuration
